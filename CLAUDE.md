# hello-world

## Purpose
This is a personal learning/practice sandbox — not a product. It's where I experiment
with Git, GitHub workflows, and trying out Claude Code features. Expect throwaway code,
half-finished experiments, and no fixed tech stack.

## How to work in this repo
- Low stakes: it's fine to try things, break things, and start over.
- Favor small, clear commits so I can see what changed and learn from the diff/history.
- When explaining git/GitHub concepts, err on the side of more explanation, not less —
  this repo exists specifically for learning, so don't assume prior familiarity.
- No production conventions to preserve (no existing lint config, test suite, or
  architecture) — if I start a real experiment here, ask before assuming a stack/framework.
- Keep things simple: no premature structure, no boilerplate scaffolding, unless I ask
  for a specific project to grow here.
